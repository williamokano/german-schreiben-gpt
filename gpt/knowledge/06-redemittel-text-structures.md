# Redemittel & Text Structures

Reference document for Deutsch Schreibtrainer. Skeletons for every supported text type, phrase banks by function and level, connector progressions, and register transformations. Offer these as *scaffolding the student assembles*, never as text you write for them — point to the skeleton, let them fill it.

## 1. Text-Type Skeletons

### Formelle E-Mail / formeller Brief (Anfrage, Mitteilung)
1. **Betreff** — konkret, 3–7 Wörter („Reparatur der Heizung, Wohnung 12")
2. **Anrede** — Sehr geehrte Frau X, / Sehr geehrter Herr Y, / Sehr geehrte Damen und Herren,
3. **Schreibanlass** — warum schreibe ich? (1–2 Sätze)
4. **Hauptteil** — Inhaltspunkte in logischer Reihenfolge, je 1–2 Sätze, Absätze nutzen
5. **Bitte/Erwartung** — was soll passieren? (höflich, KII)
6. **Abschlusssatz + Gruß** — Mit freundlichen Grüßen + Vor- und Nachname

### Beschwerde / Reklamation (B1+)
Betreff → Anrede → sachlicher Anlass mit Details (Datum, Rechnungsnummer!) → Problem konkret beschreiben → bisherige Schritte („Ich habe bereits …") → Forderung/Erwartung mit Frist → höflich-bestimmter Schluss („Ich erwarte Ihre Antwort bis …" / B2+: „Sollte ich bis … nichts hören, …"). Ton: bestimmt, nie beleidigend — Ärger zeigen, Respekt behalten.

### Informelle E-Mail / Nachricht
Anrede (Liebe/r X, / Hallo X,) → Reaktion/Dank („Schön, von dir zu hören!") → Neuigkeiten/Antworten auf Fragen → eigene Fragen/Vorschlag → Vorfreude/Wunsch → Gruß (Liebe Grüße / Bis bald / LG) + Vorname.

### Forumsbeitrag / Meinungstext (Goethe B1 Teil 2, B2 Teil 1)
Aufhänger (das Thema + warum es mich betrifft) → Meinung klar benennen → Begründung 1 mit Beispiel aus eigener Erfahrung → ggf. Gegenseite kurz anerkennen (B2+: Konzession!) → Fazit/Appell. Kein Betreff, keine Grußformel — aber ein Titel, wenn gefordert.

### Bewerbungsanschreiben (B1+)
Betreff mit Stellenbezug → Anrede (namentlich, wenn bekannt) → Einstieg mit Bezug zur Anzeige (nicht „Hiermit bewerbe ich mich" als einzige Idee — besser: was reizt mich + was bringe ich) → Qualifikationen mit Belegen (nicht nur behaupten) → Motivation für genau diese Firma → Verfügbarkeit → Gesprächswunsch → MfG.

### Erörterung / Stellungnahme (B2/C1, telc C1, TestDaF Teil 2)
Einleitung (Thema + Relevanz + These/Fragestellung) → Argumentationsteil: Pro-Argumente mit Beispielen, Contra ernst nehmen, entkräften oder abwägen → eigene begründete Position → Schluss (Zusammenfassung + Ausblick/Forderung). Absätze = Argumente. Ein Argument pro Absatz, Topic Sentence zuerst.

### Grafikbeschreibung (TestDaF)
Einleitungssatz (Was zeigt die Grafik? Quelle, Zeitraum, Einheit) → 2–3 **ausgewählte** auffällige Daten mit Vergleichen (nicht alles aufzählen!) → auffälligste Entwicklung benennen → Überleitung zur Argumentation („Diese Entwicklung wirft die Frage auf, …").

### Entschuldigung / Absage (halbformell)
Anrede → Dank/Bezug → Absage/Entschuldigung klar, aber früh im Text → Grund (ehrlich, angemessen dosiert) → Bedauern → Alternative/Wiedergutmachung anbieten → Gruß. Diplomatie-Kern: erst danken, dann absagen, immer Alternative.

## 2. Phrase Banks by Function and Level

Levels are floors: everything at A2 stays available at C1. Teach students to *own* one phrase per function per register before collecting more.

### Anlass nennen (formell)
| Level | Redemittel |
|---|---|
| A2/B1 | Ich schreibe Ihnen, weil … · Ich habe eine Frage zu … |
| B1 | Ich wende mich an Sie, weil … · Ich habe Ihre Anzeige gelesen und … |
| B2+ | Bezug nehmend auf … · Anlass meines Schreibens ist … · Mit großem Interesse habe ich … gelesen |

### Bitten & Fordern
| Level | Redemittel |
|---|---|
| A2 | Können Sie bitte …? · Ich möchte gern … |
| B1 | Könnten Sie …? · Ich hätte gern … · Wäre es möglich, …? · Ich bitte Sie, … zu … |
| B2 | Ich wäre Ihnen dankbar/verbunden, wenn … · Ich bitte Sie daher, … · Ich fordere Sie auf, … (Beschwerde) |
| C1 | Ich möchte Sie bitten, dafür Sorge zu tragen, dass … · Ich sehe mich gezwungen, … (Eskalation) |

### Meinung äußern
| Level | Redemittel |
|---|---|
| B1 | Ich finde, dass … · Meiner Meinung nach … · Ich bin der Meinung, dass … |
| B2 | Ich bin davon überzeugt, dass … · Aus meiner Sicht … · Ich stehe auf dem Standpunkt, dass … |
| C1 | Meines Erachtens … · Es lässt sich kaum bestreiten, dass … · Ich neige zu der Auffassung, dass … |

### Konzession & Gegenargument (B2+ Kernkompetenz)
Zwar …, aber … · Es stimmt, dass …, dennoch … · Einerseits …, andererseits … · Man könnte einwenden, dass … — allerdings … · So berechtigt dieser Einwand ist, … (C1) · Dem ist entgegenzuhalten, dass … (C1)

### Beispiele & Belege
zum Beispiel · beispielsweise · Ein Beispiel dafür ist … · Das zeigt sich etwa daran, dass … (B2) · Als Beleg mag dienen, dass … (C1) · Aus eigener Erfahrung kann ich sagen, dass … (Forumsbeitrag)

### Dank & Reaktion (informell / formell)
| informell | formell |
|---|---|
| Danke für deine Nachricht! | Vielen Dank für Ihre E-Mail vom … |
| Schön, von dir zu hören! | Ich freue mich über Ihre schnelle Antwort. |
| Sorry, dass ich erst jetzt antworte! | Bitte entschuldigen Sie meine späte Antwort. |

### Vorschlagen & Aushandeln
Wollen wir …? (A2) · Wie wäre es, wenn …? (B1) · Ich schlage vor, dass … (B1) · Wir könnten uns darauf einigen, dass … (B2) · Als Kompromiss käme infrage, … (C1)

### Abschluss & Ausblick (formell)
Ich freue mich auf Ihre Antwort. (B1) · Über eine baldige Rückmeldung würde ich mich freuen. (B1/B2) · Für Rückfragen stehe ich Ihnen gern zur Verfügung. (B2) · Ich danke Ihnen im Voraus für Ihre Mühe. (B2)

## 3. Connector Progression

| Funktion | A1–A2 | B1 | B2 | C1 |
|---|---|---|---|---|
| Reihung | und, auch, dann | außerdem, danach, anschließend | darüber hinaus, zudem | überdies, nicht zuletzt |
| Grund | weil, denn | deshalb, deswegen, da | daher, aus diesem Grund | infolgedessen, zumal |
| Gegensatz | aber | trotzdem, obwohl | dennoch, allerdings, hingegen | gleichwohl, wohingegen, indes |
| Bedingung | wenn | falls | sofern, es sei denn | vorausgesetzt, dass |
| Zweck | — | damit, um … zu | — | mit dem Ziel, … zu |
| Folge | — | also, deshalb | folglich, somit | demzufolge, mithin |
| Zeit | dann, zuerst | während, als, bevor, nachdem | sobald, seitdem | ehe, kaum dass |

Coaching rule: a text living entirely in column A1–A2 at B1 level → KON pattern; one column above session level per text is a great Mikro-Challenge.

## 4. Register Transformation Table

The same act in three registers — the fastest way to teach register as *choice*, not rule:

| Sprechakt | informell (du) | neutral/halbformell | formell (Sie) |
|---|---|---|---|
| Bitte | Kannst du mir kurz helfen? | Könnten Sie mir bitte helfen? | Ich wäre Ihnen sehr dankbar, wenn Sie mir behilflich sein könnten. |
| Absage | Sorry, ich schaff's nicht! 😕 | Leider kann ich nicht kommen. | Leider muss ich Ihnen mitteilen, dass ich den Termin nicht wahrnehmen kann. |
| Beschwerde | Das nervt echt langsam! | Leider gibt es schon wieder ein Problem. | Ich muss Ihnen leider mitteilen, dass das Problem weiterhin besteht. |
| Erinnerung | Denkst du noch an …? | Ich wollte kurz nachfragen, ob … | Ich möchte Sie höflich an … erinnern. |
| Zusage | Klar, bin dabei! | Ja, das passt mir gut. | Gerne bestätige ich Ihnen den Termin am … |
| Dank | Tausend Dank, du bist die Beste! | Vielen Dank für Ihre Hilfe. | Für Ihre Unterstützung danke ich Ihnen herzlich. |

## 5. Anrede & Grußformel Matrix

| Kontext | Anrede | Gruß |
|---|---|---|
| Enge Freunde | Hallo Anna! / Liebe Anna, / Hi! | Liebe Grüße / LG / Bis bald! |
| Bekannte, Kollegen (du) | Hallo Anna, / Lieber Tom, | Viele Grüße / Herzliche Grüße |
| Kollegen, Kursleitung (Sie, bekannt) | Liebe Frau Johannsen, / Guten Tag Herr Fontes, | Viele Grüße / Beste Grüße |
| Formell, Name bekannt | Sehr geehrte Frau Demir, | Mit freundlichen Grüßen |
| Formell, Name unbekannt | Sehr geehrte Damen und Herren, | Mit freundlichen Grüßen |

Fehlerklassiker (R/O): nach der Anrede folgt ein Komma und der Text beginnt **klein** (außer bei Nomen/Sie). „Mit freundlichen Grüßen" hat **kein Komma** danach, nur den Namen darunter.

## 6. How to Deploy This File in Coaching

- Student stuck starting a text → give the skeleton for the text type, not sentences. „Hier ist das Gerüst einer Beschwerde — füll es mit deinem Fall."
- Student uses the same phrase repeatedly → offer the function table for their level and one row above; let them choose.
- Register errors (R) → show the transformation table row for that speech act; have them re-render their own sentence.
- Exam prep → require students to *own* (produce from memory) one formal opener, one KII request, one concession pattern, and one closing — those four appear in nearly every exam letter.
- Never hand over a fully pre-written letter assembled from this file. Scaffolding, not ghostwriting.
